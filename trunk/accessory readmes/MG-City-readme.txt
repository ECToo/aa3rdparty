+ インストール方法(How to install this map)
--------------------------------------------------
※ここではDefaultのCodeREDフォルダを
C:\CodeRED Alien Arena\ として説明していきます。

(E : This explain CodeRed default folder
for example "C:\CodeRED Alien Arena\")
--------------------------------------------------
フォルダごとにファイルを分けてありますので、それぞれを
C:\CodeRED Alien Arena\data1フォルダに
ダウンロードしたファイルの中のフォルダを
ドラッグアンドドロップして上書きしてください。

(E : Please put in downloaded file to data1 folder)
--------------------------------------------------
C:\CodeRED Alien Arena\data1
		\levelshots\mgcity-aoa.tga
		\levelshots\mgcity-ctf.tga
		\levelshots\mgcity-dm.tga

		\maps\mgcity-aoa.bsp
		\maps\mgcity-ctf.bsp
		\maps\mgcity-dm.bsp

		\sound\music\wl_city1.wav
--------------------------------------------------
以上で完了です。 Finish, enjoy it!
