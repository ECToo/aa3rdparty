Fort Of Pillars for ALIEN ARENA 2007

TYPE: CTF & DM

Made by Michael Allison (aka Michael_TSM) 

A small tight, fast and explosive capture the flag map, that can also be used as a deathmatch map.

INSTALLTION:
- Extract contents of this .rar file to your Alien Arena directory
- When the dialogue box appears click 'yes to all'

RUNNING THE MAP, AS CAPTURE THE FLAG:
- Run AA 2007, once loaded press the ~ button
- Wait for map to load, then press ~ again, and type "set ctf 1" into the console
- Then type into the command console 'map ctf-fortofpillars' 
- Press ~ again to get rid of the command console
- enjoy

RUNNING THE MAP, AS A DEATHMATCH:
- Run AA 2007, once loaded press the ~ button
- Wait for map to load, then press ~ again
- Type into the command console 'map dm-fortofpillars' 
- Press ~ again to get rid of the command console
- enjoy

ALTERNATIVELY:
- You can edit the maps.lst file found in the 'data1' folder which is in your base Alien Arena directory.
- Open the file in notepad, and in there repspective area's type in 
	ctf-fortofpillars "CTF Fort of Pillars"
	dm-fortofpillars "DM Fort of Pillars"
- Save the file
- Now you can start the maps anytime by simply going to the "multi" selection in Alien Arena's main menu.
