  Scarecrow fiels is a small farm that was abandoned by their previous owners recently. Victim of abandonment and looting, this quiet and lonely territory has been conditioned and adapted to host bloody and murderous confrontations between Aliens, Humans and Machines. What would be better? Give it this use? Or stay in oblivion? The decision is up to you.
       
Game rules DM (Deathmatch) 2-14 Players
CTF (Capture The Flag) 4-14 Players
AOA (All Out Assault) 2-10 Players

TCA (Team Core Assault) 4-14 Players

Custom environment and skyboxes.

Installation It is always recommended to back up files or folders to be modified, in this case of Alien Arena folder.

To install the maps unzip the *. Zip and copy or move the "Arena" folders and "docs" in the root folder of Alien Arena replacing existing folders.

If you installed the version 1.0 map (the first version), and you are only updaing it, there are some files that have changed course. To keep clean Alien Arena folder, and not have unnecessary files that can not be used, you should delete the following files:

  • gfx/grass2.tga
  • gfx/grass4.tga
  • gfx/grass4s.tga
  • gfx/grass5s.tga
  • gfx/rain02a.tga
  • gfx/rain02b.tga
  • gfx/rain02ba.tga
  • gfx/rain02bb.tga
  • gfx/rain02bc.tga
  • gfx/rain02bd.tga
  • gfx/rain02be.tga
  • gfx/rain02c.tga
  • gfx/rain02d.tga
  • gfx/rain02e.tga
  • gfx/rain03.tga
  • gfx/rain03b.tga
  • gfx/maps/scarecrow_rfx.tga
  • gfx/maps/scarecrow_rfx2.tga
  • gfx/maps/ (Only if this folder is empty after deleting the previous two files)

Once installed, the maps can change the height of the tree branches only in version 1.1. To do this, look for the *.rscript, which are located in scripts/maps, and they are the next:

  • scripts/maps/cp-scarecrow.rscript
  • scripts/maps/ctf-scarecrow.rscript
  • scripts/maps/db-scarecrow.rscript
  • scripts/maps/dm-scarecrow.rscript
  • scripts/maps/tca-scarecrow.rscript

These files can be opened with a plain text editor such as Notepad, and once open, look called "textures/blanks/blank2" line (one for each file) and replace them with "textures/blanks/blank3". Now the tree branches appear higher.

If you want to uninstall the maps, the files used by them, are available clicking here. There are files that are shared with other game maps (They come with an asterisk * at the end), so if all the files are deleted, could have problems with other foreign maps.

Comments Scarecrow Fields is my first map.

My idea was to change the style of science fiction, aliens, and spacecraft that have Alien Arena for something different, more common and more realistic. For that reason I have not taken resources (textures, sounds, ...) of the game.

It is a fairly simple map because not have programming, is artistically beautiful, has good lighting and sound effects and environmental. It is a fun map, although it is not labyrinth, with many possibilities thanks to the amount of game modes offered, including Cattle Prod (CP). Each game mode has a different climate.

It has no known bug, and has been optimized to work the best as possible. But you have to know that it has been tested with bots, and if there are more players than indicated in each game mode, It can be problems and Alien Arena can crash.

I would like to say thanks to Dakota, Max, Freaky, BetelGeuse, Irritant, and all Alien Arena community (http://red.planetarena.org/forum/index.php) for helping me with my problems and doubts, without them there would surely have ended this map. So thanks for your interest.

Hope you all like it and enjoy it...
