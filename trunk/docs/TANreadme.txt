DM - Tangle for ALIEN ARENA 2007

TYPE: DM

Made by Michael Allison (aka Michael_TSM) 

An unusual but simple deathmatch arena,

INSTALLTION:
- Extract contents of this .rar file to your Alien Arena directory
- When the dialogue box appears click 'yes to all'
- It is advisable to turn the bloom effect of in your video settings, as this will make the map look poor.

RUNNING THE MAP, AS A DEATHMATCH:
- Run AA 2007, once loaded press the esc button
- Type into the command console 'map dm-tangle' 
- Press ~ again to get rid of the command console
- enjoy

ALTERNATIVELY:
- You can edit the maps.lst file found in the 'data1' folder which is in your base Alien Arena directory.
- Open the file in notepad, and in there repspective area's type in 
	dm-tangle "DM Tangle"
- Save the file
- Now you can start the maps anytime by simply going to the "multi" selection in Alien Arena's main menu.
