These are only a guide, some packages may be named differently between distributions.

Ubuntu and Debian based systems:

sudo apt-get install subversion git-core libglib2.0-dev libxml2-dev libpng12-dev mesa-common-dev libgtk2.0-dev libgtkglext1-dev


Fedora and RPM based systems:

su -c 'yum install subversion wget git glib2-devel libxml2-devel libpng-devel mesa-libGL-devel gtk2-devel gtkglext-devel wget'
